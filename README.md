# 🚀 AI Viral Content Predictor

An AI-powered web application that predicts whether a YouTube video will go viral using Machine Learning and data analytics.

---

## 🎯 Features

- 🔗 Analyze any YouTube video via link
- 📊 Fetch real-time data (views, likes, comments)
- 🤖 AI-based viral prediction
- 🧠 Explainable AI insights
- 🏷️ Smart hashtag generator
- ✨ AI title generator
- 📈 Graph-based visualization
- 🔍 Video comparison system

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Machine Learning (Scikit-learn)
- YouTube Data API
- Pandas, NumPy
- Matplotlib / Plotly
